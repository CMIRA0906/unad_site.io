Selenium WebDriver




Hablemos un poco sobre la herramienta Top 1 en automatización de pruebas de software:

Selenium WebDrier:

Es una herramienta de automatización que cuenta con un API(Application Programming Interface) orientada a objetos la cual permite realizar la automatización de pruebas para aplicaciones web. Su modelo de trabajo consiste en recibir comandos enviados a través de un API cliente, estos son enviados al navegador donde se ejecutan las operaciones y finalmente obtienen los resultados de dicha operación. Selenium Web Driver hace llamadas directas al navegador usando el soporte nativo de automatización de cada uno de ellos.
